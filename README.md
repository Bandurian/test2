# MealHub - Expo Native App

Нативное мобильное приложение для планирования питания, управления рецептами и создания списков покупок.

## Функционал

- **Аутентификация** - регистрация и вход через Supabase
- **Рецепты** - просмотр, создание и управление рецептами
- **Планировщик питания** - планирование приемов пищи на день
- **Список покупок** - автоматическая генерация и управление списком покупок
- **Профиль** - управление личными данными и предпочтениями питания

## Технологии

- **Expo** - платформа для разработки мобильных приложений
- **React Native** - фреймворк для создания нативных приложений
- **Expo Router** - файловая маршрутизация
- **Supabase** - база данных и аутентификация
- **TypeScript** - типизация

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. Используйте Expo Go на телефоне или эмулятор:
- iOS: нажмите `i` для запуска в iOS симуляторе
- Android: нажмите `a` для запуска в Android эмуляторе
- Или сканируйте QR код в приложении Expo Go

## Структура проекта

```
app/
  (auth)/          - экраны аутентификации
  (tabs)/          - основные экраны с табами
    index.tsx      - экран рецептов
    planner.tsx    - планировщик питания
    shopping.tsx   - список покупок
    profile.tsx    - профиль пользователя
lib/
  supabase/        - конфигурация Supabase
constants/         - константы и темы
hooks/             - кастомные хуки
components/        - переиспользуемые компоненты
```

## База данных

Приложение использует Supabase со следующими таблицами:
- `profiles` - профили пользователей
- `recipes` - рецепты
- `recipe_ingredients` - ингредиенты рецептов
- `recipe_steps` - шаги приготовления
- `meal_plans` - планы питания
- `meal_plan_entries` - записи в планах
- `shopping_lists` - списки покупок
- `shopping_list_items` - товары в списках
